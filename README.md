# SSPortal ニュースティッカーウィジェット

RSSフィードからニュースを取得し、マーキー/カルーセル形式で表示するウィジェット。

## 機能

- **複数RSSフィード対応**: 複数のRSSフィードを登録・管理
- **2つの表示モード**:
  - マーキー: 横スクロールで連続表示
  - カルーセル: 一定間隔で切り替え表示
- **カスタマイズ可能**:
  - 表示順序（時間順/ランダム）
  - 表示内容（タイトル/説明文）
  - フォント・サイズ・色
  - 背景・ボーダー

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
# http://localhost:3001 でプレビュー
```

## ファイル構成

```
├── @ssportal/          # SDK（編集禁止）
├── src/
│   ├── manifest.json   # メタデータ
│   ├── index.tsx       # ウィジェット本体
│   ├── config.tsx      # 設定パネル
│   └── types.ts        # 型定義
├── app/
│   └── page.tsx        # プレビュー画面
└── package.json
```

## 設定タブ

| タブ | 内容 |
|------|------|
| フィード | RSSフィード一覧管理（追加/削除/有効無効） |
| 表示 | ティッカータイプ、速度、順序、表示内容 |
| 文字 | フォント、サイズ、位置、色 |
| 設定 | 背景色、ボーダー |
